# US-002 · Themes page & Sub-themes page

## User story

**As a** user
**I want to** browse the 8 TNCOOP themes and drill into their sub-themes
**So that** I can find and select the indicator I want to analyse

## Acceptance criteria

- [ ] ThemesPage: responsive grid of 8 theme cards (2 cols desktop, 1 col mobile)
- [ ] Each theme card: icon, name, sub-theme count, data availability badge
- [ ] Themes without sub-themes: "Coming soon" overlay, not clickable
- [ ] Click active theme → navigates to `/theme/:themeId`
- [ ] SubThemesPage: header with back link + theme name, list of sub-theme cards
- [ ] Sub-theme cards grouped visually, each showing: name, indicator count, data status, description snippet
- [ ] Sub-themes without data: "Data pending" badge, not clickable
- [ ] Click active sub-theme → navigates to `/theme/:themeId/sub/:subId`
- [ ] Full-width layout (no sidebar), max-width 1000px centered
